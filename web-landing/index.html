How to install the APK

ScrollIQ isn’t on Google Play yet, so you install it directly. Takes about a minute.

  1. 1
    Download the app. Tap Download for Android above and wait for scrolliq.apk to finish downloading.
  2. 2
    Open the file. Open it from your notification shade or your Files → Downloads folder.
  3. 3
    Allow install from this source. If Android warns you, tap Settings and turn on Allow from this source for your browser or Files app, then go back.
  4. 4
    Install & open. Tap Install, then Open. Sign up and grant the usage & accessibility permissions so ScrollIQ can measure your Brain Score.

Safe to install: the APK is signed by ScrollIQ. On Android 8+ the “unknown source” prompt is per-app, so you’re only trusting the app you downloaded from.
